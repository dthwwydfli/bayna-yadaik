# bayna-yadaik
An Arabic vocabulary revision tool built to support self-study of the Bayna Yadayk textbook series. Designed for personal use ahead of a trip to Egypt.
Features

Flashcard-based vocabulary review
Spaced repetition for retention
Quiz mode
Egypt phrasebook
Study schedule

Built with

HTML, CSS, JavaScript (single file)

Live site
https://dthwwydfli.github.io/bayna-yadaik/
How it was made
This project was built with the assistance of Claude (Anthropic's AI), used as a coding collaborator to generate and iterate on the HTML/CSS/JS. The vocabulary content and learning structure were directed by me based on the Bayna Yadayk curriculum.
